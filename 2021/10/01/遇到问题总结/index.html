一些关于Vue2的笔记

进度条功能，进度条未到100%时，轮询请求接口

使用到：闭包、垃圾回收、Vue生命周期

闭包:函数内嵌套函数，内部函数当做返回值
在Vue beforeDestory销毁定时器，做垃圾回收，防止内存的消耗

手写节流和防抖

函数节流：规定在一个单位时间内，只能触发一次函数，如果这个单位时间内触发多次函数，只有一次生效； 典型的案例就是鼠标不断点击触发，规定在n秒内多次点击只有一次生效
函数防抖：在事件被触发n秒后再执行回调，如果在这n秒内又被触发，则重新计时；典型的案例就是输入搜索：输入结束后n秒才进行搜索请求，n秒内又输入的内容，就重新计时

手写防抖：
function debounce(fn,delay=1000){
let timer
return function(){
let args = arguments
let context = this
clearTimeout(timer)
timer = setTimeout(()=>{
fn.apply(context,args)
timer = null
},delay)
}
}

function throttle(fn,delay=1000){
let timer
return function(){
let args = arguments
let context = this
if(!timer){
timer = setTimeout(()=>{
fn.apply(context,args)
timer = null
},delay)
}
}
}
